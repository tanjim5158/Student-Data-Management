# Student-Data-Management
 Student Management System in C
A simple console-based Student Management System built in C that stores and manages student records using binary file handling.
 Features

Add Student — Enter student name, roll number, and marks for 3 subjects. Average is auto-calculated and saved.
View All Students — Display all stored student records with marks and average.
Search by Roll Number — Quickly find a specific student using their roll number.
Persistent Storage — All data is saved to a binary file so records remain after the program closes.

🛠️ Concepts Used

Structs (struct student)
Binary File I/O (fread, fwrite)
Arrays and loops
Switch-case menu system
Input handling with scanf

How to Run
Prerequisites

GCC compiler (or any C compiler)
